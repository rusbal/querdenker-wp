== Description ==

With this plugin you can add the Google Analytics Code in no time to your header or footers.


== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Choose your options via "Settings -> Evolution Google Analytics"
